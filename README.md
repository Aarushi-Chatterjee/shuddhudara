# 🌱 SHUDDHUDARA & PurePulse

**The Future of Plant-Based Clean Air & Tech-Eco Innovation**

SHUDDHUDARA is a comprehensive platform dedicated to advancing sustainable air purification through the synergy of natural ecosystems and modern technology. Our flagship initiatives represent a new era of environmental restoration and community engagement.

Developed in collaboration with **Vidaverde International** and supported by the **Youth Development for Climate Tech (YDfCT)** program powered by **Meta AI**.

## 🛠️ Technology Stack

### Frontend Excellence
- **HTML5 & CSS3**: Custom design systems with glassmorphism and aero-tech aesthetics.
- **Vanilla JavaScript**: High-performance interactions and real-time state management.
- **Typography**: `Syne`, `Plus Jakarta Sans`, and `Poppins` for a premium feel.

### Robust Backend
- **Node.js**: Asynchronous event-driven runtime.
- **Express.js**: RESTful API architecture.
- **PostgreSQL (Neon)**: Serverless SQL data persistence.

### Security & Authentication
- **bcryptjs**: Advanced password hashing.
- **JWT**: Secure token-based session management.
- **pg**: PostgreSQL client for Node.js.

## 🚀 Installation & Setup

1. **Clone the Repository**
   ```bash
   cd C:\Users\HP\.gemini\antigravity\scratch\shuddhudara
   ```

2. **Backend Configuration**
   ```bash
   cd backend
   npm install
   ```
   *Create a `.env` file in the `backend` directory with your `DATABASE_URL` (Neon PostgreSQL) and `JWT_SECRET`.*

3. **Start the Engine**
   ```bash
   # Run the server
   npm run dev
   ```

4. **Launch Interface**
   Open `frontend/home/index.html` in your favorite modern browser.
